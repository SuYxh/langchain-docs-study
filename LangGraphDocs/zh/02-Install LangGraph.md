# 安装 LangGraph

要安装基础 LangGraph 包：

<CodeGroup>
  ```bash npm theme={null}
  npm install @langchain/langgraph @langchain/core
  ```

  ```bash pnpm theme={null}
  pnpm add @langchain/langgraph @langchain/core
  ```

  ```bash yarn theme={null}
  yarn add @langchain/langgraph @langchain/core
  ```

  ```bash bun theme={null}
  bun add @langchain/langgraph @langchain/core
  ```
</CodeGroup>

要使用 LangGraph，你通常还需要访问大语言模型并定义工具。
你可以用任何你认为合适的方式来实现这一点。

一种方式（我们会在文档中使用）是使用 [LangChain](/oss/javascript/langchain/overview)。

使用以下命令安装 LangChain：

<CodeGroup>
  ```bash npm theme={null}
  npm install langchain
  ```

  ```bash pnpm theme={null}
  pnpm add langchain
  ```

  ```bash yarn theme={null}
  yarn add langchain
  ```

  ```bash bun theme={null}
  bun add langchain
  ```
</CodeGroup>

如需使用特定 LLM 提供商的包，你需要单独安装它们。

有关提供商特定的安装说明，请参阅 [集成](/oss/javascript/integrations/providers/overview) 页面。
